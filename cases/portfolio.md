## The Problem

Most PM portfolios are static and forgettable. I wanted something that demonstrates how I think — interactive, opinionated, and built with the same tools I advocate for.

## What I Did

- Designed the entire UX: collaboration mode, draggable elements, cursor tour, canvas-style layout
- Built with Claude Code — vanilla HTML/CSS/JS, no framework
- Integrated Obsidian-backed blog with auto-publish pipeline
- Dark mode, responsive, accessible — all shipped in days, not weeks

## Results

Shipped v1 to v3 in under a week. Living proof that non-technical founders can build production-quality products with AI.
